# 📊 LocalSense BI — Offline Business Intelligence Dashboard

**100% Client-Side. Zero Server. Absolute Privacy.**

LocalSense BI is a powerful, drag-and-drop Business Intelligence dashboard that runs entirely in your browser. Drop your CSV, JSON, or Excel file and get instant KPIs, interactive charts, and filterable data—all without a single byte leaving your machine.

## ✨ Features

- **📂 Multi-Format Support:** CSV, JSON, Excel (.xlsx/.xls)
- **📊 7 Chart Types:** Line, Bar, Pie, Doughnut, Polar Area, Radar, Scatter
- **🎯 Smart KPI Cards:** Sum, Average, Max, Count with column selector
- **🔍 Interactive Slicers:** Filter data in real-time from the sidebar
- **🧠 Auto-Detection:** Automatically identifies data types (Number, Date, Email, etc.)
- **🔄 Drag & Drop Dashboard:** Resize and rearrange charts with GridStack.js
- **✨ Auto Dashboard:** One-click intelligent chart generation
- **🌓 Dark/Light Theme:** Full theme support
- **📥 Export:** Excel, PNG (full dashboard), PDF (Pro version)
- **🔒 100% Offline:** No server, no API calls, no tracking

---

## 🛠️ Tech Stack

- **Tailwind CSS** — Styling
- **GridStack.js** — Drag & drop dashboard layout
- **Chart.js** — Interactive charts
- **PapaParse** — CSV parsing
- **SheetJS (xlsx)** — Excel parsing & export
- **Dexie.js** — IndexedDB wrapper (Pro version)
